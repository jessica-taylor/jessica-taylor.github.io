For the Stanford hackathon (May 8, 2011), I created a reversible programming language. This means that all functions of the program can be inverted. For example, if you wrote a function to convert an integer to a string, you would not need an additional function to convert the string to an integer. See the Lisp source code for the source and examples of reversible programs. All the code was written on May 8 except comments.
